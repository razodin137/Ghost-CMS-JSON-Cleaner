# Importing into Directus

This guide explains how to import the `cleaned_data.json` file generated by `clean.js` into Directus.

## Prerequisites

- Access to a Directus instance.
- An existing collection (e.g., `posts`) to import data into.

## Step 1: Create Collection Schema

Ensure your Directus collection has the following fields to match the cleaned data:

| Field Name | Key | Type | Interface | Notes |
| :--- | :--- | :--- | :--- | :--- |
| **Title** | `title` | String | Input | |
| **Slug** | `slug` | String | Input | |
| **Status** | `status` | String | Dropdown | Options: `published`, `draft` |
| **Type** | `type` | String | Dropdown | Options: `post`, `page` |
| **Featured** | `featured` | Boolean | Toggle | |
| **Content** | `html` | Text | WYSIWYG | |
| **Excerpt** | `excerpt` | Text | Text Area | |
| **Published Date** | `published_at` | DateTime | DateTime | |
| **Created Date** | `created_at` | DateTime | DateTime | |
| **Updated Date** | `updated_at` | DateTime | DateTime | |
| **Tags** | `tags` | JSON / CSV | Tags | Maps to array of strings |
| **Author** | `author` | String | Input | Name string |
| **Feature Image** | `feature_image_url` | String | Input | URL text only |

## Step 2: Import Data

1.  In Directus, navigate to your Collection (e.g., **Posts**).
2.  Click the **Import / Export** arrow in the sidebar (usually top right).
3.  Select **Import from File**.
4.  Upload the `cleaned_data.json` file.
5.  Directus will map the JSON keys to your collection fields. Verify the mapping.
6.  Click **Import**.

## Notes

- **Images**: This tool does **not** upload images to Directus files. It only preserves the original image URL in the `feature_image_url` field.
- **Relational Data**: Tags and Authors are imported as simple values (arrays or strings). If you need deep relational linking (e.g. to a `Users` collection), you may need to post-process the data in Directus.
